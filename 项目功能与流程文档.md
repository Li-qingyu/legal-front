# 项目功能与流程文档

## 1. 管理端功能与流程

### 1.1 管理端概述
管理端是系统的后台管理界面，主要用于管理员对系统内容进行管理和维护，包括用户管理、法律类型管理、法律案例管理、法律书管理和法律条文管理等功能。

### 1.2 管理端功能模块

#### 1.2.1 用户管理
- **功能描述**：管理系统用户，包括查看用户列表、添加新用户、编辑用户信息、删除用户以及启用/禁用用户账号。
- **流程**：
  1. 管理员登录管理端
  2. 进入用户管理页面
  3. 查看用户列表，可通过搜索筛选特定用户
  4. 点击添加按钮，填写用户信息并提交
  5. 点击编辑按钮，修改用户信息并保存
  6. 点击删除按钮，确认后删除用户
  7. 点击启用/禁用按钮，修改用户状态

#### 1.2.2 法律类型管理
- **功能描述**：管理法律类型，包括查看类型列表、添加新的法律类型、编辑类型信息以及删除法律类型。
- **流程**：
  1. 管理员登录管理端
  2. 进入法律类型管理页面
  3. 查看类型列表，可通过搜索筛选特定类型
  4. 点击添加按钮，填写类型信息并提交
  5. 点击编辑按钮，修改类型信息并保存
  6. 点击删除按钮，确认后删除类型

#### 1.2.3 法律案例管理
- **功能描述**：管理法律案例，包括查看案例列表、添加新的法律案例、编辑案例信息、删除案例以及发布/下架案例。
- **流程**：
  1. 管理员登录管理端
  2. 进入法律案例管理页面
  3. 查看案例列表，可通过搜索和筛选查找特定案例
  4. 点击添加按钮，填写案例信息（标题、内容、分类等）并提交
  5. 点击编辑按钮，修改案例信息并保存
  6. 点击删除按钮，确认后删除案例
  7. 点击发布/下架按钮，修改案例状态

#### 1.2.4 法律书管理
- **功能描述**：管理法律书，包括查看法律书列表、添加新的法律书、编辑法律书信息以及删除法律书。
- **流程**：
  1. 管理员登录管理端
  2. 进入法律书管理页面
  3. 查看法律书列表，可通过搜索和筛选查找特定法律书
  4. 点击添加按钮，填写法律书信息并提交
  5. 点击编辑按钮，修改法律书信息并保存
  6. 点击删除按钮，确认后删除法律书

#### 1.2.5 法律条文管理
- **功能描述**：管理法律条文，包括查看法律条文列表、添加新的法律条文、编辑法律条文信息以及删除法律条文。
- **流程**：
  1. 管理员登录管理端
  2. 进入法律条文管理页面
  3. 查看法律条文列表，可通过搜索和筛选查找特定条文
  4. 点击添加按钮，填写法律条文信息并提交
  5. 点击编辑按钮，修改法律条文信息并保存
  6. 点击删除按钮，确认后删除法律条文

### 1.3 管理端技术实现
- **前端**：使用Vue 3 + Element Plus构建管理端界面
- **后端**：使用Spring Boot提供API接口
- **认证**：使用JWT进行身份验证
- **权限控制**：基于角色的访问控制

## 2. AI咨询流程描述

### 2.1 AI咨询概述
AI咨询模块是系统的核心功能之一，利用Spring AI框架结合阿里云百炼平台的Qwen模型，为用户提供专业的法律咨询服务。

### 2.2 AI咨询技术架构
- **AI框架**：Spring AI
- **AI模型**：阿里云百炼平台的Qwen模型（通过Spring AI的OpenAI接口调用）
- **工具集成**：Spring AI的tools功能，实现数据库查询能力
- **知识库**：系统数据库中的法律案例和法律条文

### 2.3 AI咨询流程
1. **用户提交问题**：用户在前端界面输入法律咨询问题
2. **请求处理**：后端接收用户请求，进行初步处理
3. **工具调用**：使用Spring AI的tools功能
   - 查询数据库中的相关法律案例
   - 查询相关法律条文和知识
4. **AI模型调用**：将用户问题和查询结果作为输入，调用阿里云百炼平台的Qwen模型
5. **生成回答**：AI模型结合法律知识生成专业的法律回答
6. **结果返回**：将AI回答返回给前端用户
7. **用户评价**：用户可以对AI回答进行评价（1-5星）
8. **历史记录**：系统保存咨询历史，用户可在个人中心查看

### 2.4 AI咨询技术实现
- **Spring AI集成**：配置Spring AI框架，连接阿里云百炼平台
- **工具开发**：开发Spring AI的tools功能，实现数据库查询能力
- **知识库管理**：通过管理端维护法律案例和法律条文数据库
- **性能优化**：使用异步处理AI咨询请求，提高系统响应速度
- **质量控制**：定期评估AI回答质量，优化模型参数

## 3. 用户端功能与流程

### 3.1 用户端概述
用户端是面向普通用户的前端界面，主要提供法律咨询、法律案例查询、法律条文查询等功能，以及用户个人中心管理。

### 3.2 用户端功能模块

#### 3.2.1 用户注册/登录
- **功能描述**：用户可以通过手机号、邮箱注册账号，使用账号密码登录系统。
- **流程**：
  1. 用户点击注册按钮
  2. 选择注册方式（手机号/邮箱）
  3. 填写注册信息并验证
  4. 注册成功后自动登录
  5. 登录时输入账号密码或使用第三方登录

#### 3.2.2 法律案例查询
- **功能描述**：用户可以浏览、搜索法律案例，查看案例详情，收藏感兴趣的案例。
- **流程**：
  1. 用户进入法律案例列表页
  2. 使用筛选条件（案例类型、地区、时间等）筛选案例
  3. 使用排序方式（热度、发布时间等）排序案例
  4. 点击案例卡片查看案例详情
  5. 查看案例内容和相关法条
  6. 点击收藏按钮收藏案例
  7. 在个人中心查看收藏的案例

#### 3.2.3 法律条文查询
- **功能描述**：用户可以浏览、搜索法律条文，查看条文详情，收藏感兴趣的条文。
- **流程**：
  1. 用户进入法律内容列表页
  2. 通过分类导航（法律类型、章节）查找法律条文
  3. 使用搜索功能搜索特定法律条文
  4. 点击条文标题查看条文详情
  5. 查看条文完整内容和相关案例
  6. 点击收藏按钮收藏条文
  7. 在个人中心查看收藏的条文

#### 3.2.4 AI咨询助手
- **功能描述**：用户可以向AI助手提交法律咨询问题，获取专业的法律回答，并查看历史咨询记录。
- **流程**：
  1. 用户进入AI咨询助手页面
  2. 输入法律咨询问题
  3. 提交问题
  4. 等待系统处理并生成回答
  5. 查看AI助手生成的法律回答
  6. 对回答进行评价（1-5星）
  7. 在个人中心查看历史咨询记录

#### 3.2.5 个人中心
- **功能描述**：用户可以管理个人信息，查看咨询历史和收藏内容，接收消息通知。
- **流程**：
  1. 用户进入个人中心
  2. 在左侧菜单选择相应功能
  3. **个人信息**：修改头像、昵称、联系方式等
  4. **咨询历史**：查看与AI助手的咨询历史
  5. **收藏管理**：查看收藏的案例和法律条文
  6. **消息通知**：查看系统通知和AI回复通知
  7. **设置**：修改密码、绑定手机号/邮箱、隐私设置等

### 3.3 用户端技术实现
- **前端**：使用Vue 3 + Element Plus构建用户端界面
- **路由**：使用Vue Router实现页面导航
- **状态管理**：使用Pinia管理全局状态
- **HTTP请求**：使用Axios与后端API交互
- **响应式设计**：适配不同屏幕尺寸
- **性能优化**：代码分割、资源压缩、图片优化等

## 4. 系统集成与数据流

### 4.1 系统架构
- **前端**：Vue 3 + Vite + Element Plus
- **后端**：Spring Boot + Spring AI
- **数据库**：MySQL
- **缓存**：Redis
- **认证**：JWT

### 4.2 数据流
1. **用户操作**：用户在前端界面进行操作（如提交咨询问题、查询案例等）
2. **API请求**：前端通过Axios向后端发送API请求
3. **后端处理**：后端接收请求，进行业务逻辑处理
4. **数据库操作**：后端与数据库交互，查询或修改数据
5. **AI处理**：对于咨询请求，后端调用Spring AI和AI模型进行处理
6. **响应返回**：后端将处理结果返回给前端
7. **前端展示**：前端接收响应，更新界面展示

### 4.3 安全措施
- **前端安全**：输入验证、XSS防护、CSRF防护
- **后端安全**：认证授权、密码加密、SQL注入防护、API访问控制
- **数据安全**：数据备份、数据加密、访问日志、异常监控

## 5. 总结

本项目通过整合管理端、AI咨询和用户端三大模块，构建了一个完整的法律咨询平台。管理端提供了全面的内容管理功能，AI咨询模块利用Spring AI和阿里云百炼平台的Qwen模型提供专业的法律回答，用户端则为用户提供了便捷的法律信息获取和咨询服务。

系统采用了现代化的技术栈，包括Vue 3、Spring Boot、Spring AI等，确保了系统的性能和可扩展性。通过合理的功能设计和流程优化，系统能够为用户提供专业、便捷的法律咨询服务，促进法律服务的普及和标准化。
