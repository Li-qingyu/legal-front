# 法律咨询项目文档

## 1. 项目概述

### 1.1 项目背景
随着社会法治意识的提高，越来越多的人需要法律咨询服务。传统的法律咨询方式存在时间成本高、地域限制等问题，线上法律咨询平台可以有效解决这些问题，为用户提供便捷、专业的法律服务。结合AI技术，可以进一步提高法律咨询的效率和可访问性。

### 1.2 项目目标
构建一个基于Vue的前端法律咨询平台，集成AI咨询助手，为用户提供专业、便捷的法律咨询服务，同时提供法律案例和法律条文的查询功能。

### 1.3 项目价值
- 为用户提供便捷的法律咨询服务
- 促进法律服务的普及和标准化
- 提高法律服务的效率和质量
- 利用AI技术降低法律服务门槛

## 2. 技术栈

### 2.1 前端技术
- **框架**：Vue 3
- **构建工具**：Vite
- **状态管理**：Pinia
- **路由**：Vue Router
- **UI组件库**：Element Plus
- **样式预处理**：SCSS
- **HTTP请求**：Axios
- **表单验证**：VeeValidate
- **日期处理**：Day.js
- **图表库**：ECharts
- **管理端UI**：Element Plus（提供完整的后台管理组件）

### 2.2 后端技术
- **语言**：Java
- **框架**：Spring Boot
- **AI框架**：Spring AI
- **AI模型**：阿里云百炼平台的Qwen模型（通过Spring AI的OpenAI接口调用）
- **数据库**：MySQL
- **缓存**：Redis（用于缓存热点信息）
- **认证**：Spring Security + JWT

## 6. 交互流程

### 6.1 用户注册/登录流程
1. 用户点击注册按钮
2. 选择注册方式（手机号/邮箱）
3. 填写注册信息并验证
4. 注册成功后自动登录
5. 登录时输入账号密码或使用第三方登录

### 6.2 AI咨询流程
1. 用户进入AI咨询助手页面
2. 输入法律咨询问题
3. 提交问题
4. 系统使用Spring AI的tools功能：
   - 查询数据库中的相关法律案例
   - 查询相关法律条文和知识
5. 结合查询结果，调用阿里云百炼平台的Qwen模型生成专业法律回答
6. 用户查看回答并评价

### 6.3 案例收藏流程
1. 用户浏览法律案例
2. 点击收藏按钮
3. 系统将案例添加到用户收藏列表
4. 用户可在个人中心查看收藏的案例

### 6.4 管理端流程
#### 6.4.1 管理端登录流程
1. 管理员访问管理端登录页面
2. 输入管理员账号和密码
3. 系统验证登录信息
4. 登录成功后进入管理端仪表盘

#### 6.4.2 用户管理流程
1. 管理员进入用户管理页面
2. 查看用户列表，可搜索和筛选
3. 点击添加按钮，填写用户信息并提交
4. 点击编辑按钮，修改用户信息并保存
5. 点击删除按钮，确认后删除用户
6. 点击启用/禁用按钮，修改用户状态

#### 6.4.3 法律类型管理流程
1. 管理员进入法律类型管理页面
2. 查看类型列表，可搜索和筛选
3. 点击添加按钮，填写类型信息并提交
4. 点击编辑按钮，修改类型信息并保存
5. 点击删除按钮，确认后删除类型

#### 6.4.4 法律案例管理流程
1. 管理员进入法律案例管理页面
2. 查看案例列表，可搜索和筛选
3. 点击添加按钮，填写案例信息并提交

5. 点击删除按钮，确认后删除案例
6. 点击发布/下架按钮，修改案例状态

#### 6.4.5 法律书管理流程

## 8. API接口设计

### 8.1 认证相关接口

#### 8.1.1 用户登录
- **接口地址**：`/api/auth/login`
- **请求方式**：POST
- **请求参数**：
  | 参数名 | 类型 | 必填 | 描述 |
  | :--- | :--- | :--- | :--- |
  | username | String | 是 | 用户名/邮箱/手机号 |
  | password | String | 是 | 密码 |
- **响应示例**：
  ```json
  {
    "code": true,
    "data": {
      "token": "jwt_token",
      "user": {
        "id": 1,
        "username": "admin",
        "email": "admin@example.com",
        "phone": "13800138000",
        "status": 1
      }
    },
    "msg": "登录成功"
  }
  ```
